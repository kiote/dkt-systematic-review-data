# Supplementary Materials: A Systematic Review of Deep Knowledge Tracing (2015–2025)

**Paper:** A Systematic Review of Deep Knowledge Tracing (2015–2025): Toward Responsible AI for Education  
**Journal:** Artificial Intelligence Review (Springer Nature)  
**Authors:** Ekaterina Krivich, Danial Hooshyar, Gustav Šír, Yeongwook Yang, Merja Bauters, Raija Hämäläinen, Tommi Kärkkäinen

## Contents

### Data Extraction
- `csv/information_extraction.csv` — Study-by-variable extraction matrix for all 84 included studies
- `csv/papers_with_categories.csv` — Model architecture categorization
- `csv/papers_rq2.csv` — Data quality analysis (RQ2)
- `csv/papers_rq3.csv` — Sequential stability analysis (RQ3)
- `csv/papers_rq4.csv` — Interpretability analysis (RQ4)
- `csv/quality_appraisal.csv` — Per-study quality appraisal scores (QA1–QA5)

### Screening Data
- `csv/Screenings - 1st Screening.csv` — Title/abstract screening scores
- `csv/Screenings - 2nd screening.csv` — Full-text screening scores

### Supplementary Documents
- `supplements/PRISMA_2020_checklist.pdf` — Completed PRISMA 2020 checklist
- `supplements/search_strategy.pdf` — Full database-specific search strings
- `supplements/quality_appraisal_supplement.pdf` — Per-study quality appraisal table
